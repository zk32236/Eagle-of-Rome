三、核心任务
任务1：实现 Province 类
文件路径：src/core/entities/entities.py（已有文件，需在其中添加 Province 类）

需求：

根据设计文档 §1.1 实现所有字段（私有）：

_province_id: int

_name: str

_total_land: int

_land_public: int（初始化 = total_land * 0.6）

_land_private: int（初始化 = total_land * 0.4）

_tax_base: int（默认0，后续由配置设置）

_grievance: int（范围0-3，默认0）

_tax_contract_id: Optional[int]（默认None）

_project_contract_id: Optional[int]（默认None）

_has_project: bool（默认False）

为每个字段提供 @property 只读访问器。

实现以下公共方法：

update_land_type(public_change: int, private_change: int) -> None：调整公/私地数量，确保非负。

bind_tax_contract(contract_id: int) -> None：绑定包税权合同，若已绑定则抛出 ValueError。

bind_project_contract(contract_id: int) -> None：绑定公共工程合同，若已绑定则抛出 ValueError，并设置 _has_project = True。

unbind_tax_contract() -> None：解绑包税权合同。

unbind_project_contract() -> None：解绑公共工程合同，并设置 _has_project = False。

set_grievance(value: int) -> None：设置民怨值，范围0-3，超出时抛出 ValueError。

任务2：创建 Contract 实体
文件路径：src/core/entities/contract.py（新文件）

需求：

定义 ContractType 枚举，包含 TAX 和 PROJECT。

实现 Contract 类，字段如下（私有）：

_contract_id: int

_contract_type: ContractType

_province_id: int

_base_cost: int

_create_turn: int

_is_active: bool（默认False）

_winner_id: Optional[int]（默认None）

_profit_base: int（默认0）

_is_under_execution: bool（默认False）

_complete_turn: Optional[int]（默认None）

为每个字段提供 @property 只读访问器。

实现以下公共方法：

mark_winner(winner_id: int, current_turn: int, profit_base: int) -> None：标记中标，激活合同。若合同已激活则抛出 ValueError。

mark_complete(current_turn: int) -> None：标记合同完成。

terminate() -> None：强制终止合同（用于异常情况）。

四、设计要点提醒
所有字段必须私有（以下划线开头），外部只能通过 @property 读取，通过公共方法修改。

方法命名：动词开头，清晰表达意图（如 bind_tax_contract，而非 setTaxContract）。

异常处理：在绑定、设置民怨等方法中，当条件不满足时抛出适当的 ValueError。

类型提示：使用 from typing import Optional, List 等提供完整类型注解。

初始化比例：land_public 和 land_private 严格按照 total_land * 0.6 和 total_land * 0.4 计算，取整（int 转换即可）。

不要修改任何原有代码（如已有 Figure 类等），只做新增。
