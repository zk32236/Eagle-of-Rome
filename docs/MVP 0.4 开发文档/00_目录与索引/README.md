# MVP 0.4 开发文档整理索引

整理日期：2026-06-20

本目录是 MVP 0.5 之前及其相邻阶段的历史开发材料集合。当前目录中实际包含 MVP 0.2、MVP 0.3、MVP 0.4、MVP 0.4.5、MVP 0.4.6、MVP 0.5 以及部分 MVP 0.7 历史资料，因此建议将它视为“早期历史档案库”，而不是严格的 MVP 0.4 单版本文档库。

## 本次整理方式

本次采用非破坏性整理：

- 不移动、不重命名原始历史文件。
- 新增本索引目录，提供文件清单、分类规则和后续整理建议。
- 后续如果需要，可以再按索引结果执行物理归档。

## 使用建议

1. 查找具体历史材料时，优先打开 `文件清单.csv`，按 `Category`、`Version`、`Name` 过滤。
2. 继续新增早期历史材料时，建议先放入对应分类目录，或至少在文件名中包含版本号、主题、日期。
3. 不建议继续把 MVP 0.7 以后的正式基线文档放入本目录；应放入独立版本目录，避免历史档案和当前基线混淆。
4. 如需进一步物理整理，可先备份本目录，再按 `Category` 迁移文件。

## 推荐命名格式

`EOR MVP <版本> - <模块/阶段> - <文档类型> - <主题> - <日期或编号>`

示例：

- `EOR MVP 0.4.5 - Forum阶段 - 开发报告 - 命令迁移 - DS012.docx`
- `EOR MVP 0.5 - 实体层 - 测试报告 - M1-1 Contract与Province.docx`
- `EOR MVP 0.7 - CLI-UI - 工作报告 - 广场阶段架构整改步骤3.docx`
