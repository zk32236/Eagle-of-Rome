核心任务：

创建独立的 Config 类：

文件路径：src/core/config.py

定义 Config 类，包含：

DEFAULTS 静态字典（完整复制原 game_state.py 中的默认配置结构）

__init__(self, config_path: Optional[str] = None)：设置 self._config 并调用 _load_from_file(config_path)

_load_from_file(self, config_path: Optional[str])：尝试加载 JSON，若失败则回退到 DEFAULTS（打印警告），成功则深合并

get(self, key: str, default=None)：实现点号路径解析（如 get("economic.base_tax")）

reload(self)：重新加载配置文件，失败则保持原配置，打印警告

to_dict(self)：返回 copy.deepcopy(self._config) 防止外部篡改

异常处理与回退：

捕获 FileNotFoundError、json.JSONDecodeError、PermissionError、TypeError

首次加载失败时 self._config = self.DEFAULTS

重载失败时保持 self._config 不变，仅打印警告
