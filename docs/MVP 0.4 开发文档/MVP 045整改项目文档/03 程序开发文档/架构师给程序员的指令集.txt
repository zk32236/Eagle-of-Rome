编写命令注册器：

文件路径：src/ui/commands/sys_registry.py

实现 CommandRegistry 类，包含：

_scan_and_register(self)：遍历目录下所有 .py 文件（排除 _ 开头的），动态导入模块，提取继承自 Command 的类，注册到内部字典（键为命令名和别名）

注册时检测冲突：若 name 或 aliases 已存在，抛出 ValueError 并终止启动

execute(self, cmd_name: str, state, args: List[str]) -> bool：查找命令类，实例化（传入 state），调用其 execute 方法，捕获异常并打印错误（确保不崩溃）

动态导入使用 importlib.util（参考设计文档伪代码）

编写最小CLI：

文件路径：src/ui/debug_cli.py

类 DebugCLI：

run(self)：while self.running: 读取输入，分割命令名和参数，调用 self.registry.execute(cmd_name, None, args)，根据返回值决定是否退出（若命令返回 False 可设置 self.running = False）

异常捕获：在 run 中使用 try-except 捕获所有异常，打印错误信息，防止单条命令崩溃导致整个CLI退出
